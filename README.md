### Running the Website Locally

You can build and inspect the website locally using the provided `Makefile`:

1. Install the required dependencies:
   ```bash
   make install
   ```
2. Build and serve the website:
   ```bash
   make serve
   ```
3. Access the website at:  
   **http://127.0.0.1:4000/**
